Get the color of the tile

The color of the tile is determined by TileInfo.color and the tile height. If showWater is true and the tile height is less than or equal to 0, the color is adjusted to show water.

Example usage:

var tile = Tile(0, 0, 0, TileInfo(0, "Rock", Color.hex("#808080")));
var color = tile.color();
print("Color: $color");

See Also:

  • TileInfo for the tile info class

Implementation

Color color({bool showWater = false}) {
  if (!showWater || height > 0) {
    return info.color;
  }
  final rgbColor = info.color.toRgbColor();

  final r = rgbColor.r * 0.2 + 0.4 * 0.4 * 256;
  final g = rgbColor.g * 0.2 + 0.5 * 0.4 * 256;
  final b = rgbColor.b * 0.2 + 1.0 * 0.4 * 256;
  return Color.rgb(r, g, b);
}
