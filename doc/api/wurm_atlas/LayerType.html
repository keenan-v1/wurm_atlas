LayerType

LayerType enum

Represents the type of a layer in the map.

This class provides constants for different types of layers that can be used in a map. Each layer type has a unique identifier and a display name.

Example usage:

LayerType type = LayerType.top;
print(type.fileName); // Output: "top_layer.map"

See also:

  • Layer, the base class for all layers in the map.
